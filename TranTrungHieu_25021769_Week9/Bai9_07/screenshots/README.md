# Thư mục chứa screenshots verification

Sau khi tạo PR test trên GitHub, lưu các ảnh sau vào đây:

- `01-pr-with-violations.png` — PR có lỗi cố ý: bot post inline comments + nút Merge bị khóa (status check đỏ + dòng "Required statuses must pass before merging").
- `02-branch-protection-rules.png` — Trang Settings → Branches hiển thị BP Rules đã cấu hình cho nhánh `main`.
- `03-pr-passed.png` — Sau khi push commit fix: status check `Checkstyle Review` xanh, nút Merge được kích hoạt lại.
